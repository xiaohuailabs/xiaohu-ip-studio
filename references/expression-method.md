# 表情方法:情境描述 + 演技锚点(双图法)

> 2026-05-31 验证定型。复杂/负面/微妙情绪(心虚/狡黠/苦/得意)这个 3D 萌系模型自己演不出,会自动拉回萌脸。解法:**情境描述给意图 + 传一张摸鱼记表情图当演技锚点让它抄**。纯写死面部肌肉(僵硬)和纯情境描述(被拉回萌脸)都不如这个。

## 双图法(生表情标准做法)

每张生图传**两张** `--reference`:
1. **长相锚点**:`characters/xiaohu/refs/xiaohu-ip-正面挥手.png`(锁齐刘海/红框眼镜/橙红毛衣背带裤/名牌/狐狸)
2. **演技锚点**:`characters/xiaohu/refs/演技锚点/` 按本张情绪挑一张(抄表情那股"演"的劲儿)

prompt 写法:
```
TWO references: FIRST = character identity (keep exactly), SECOND = expression energy to channel (NOT cheerful, the [情绪] look). Render in [皮肤] style. Scene: [情境].
```
⛔ **不写死面部**(eyes look left, eyebrow up 2cm...),只讲清**处境 + 想要的情绪**,让模型自己演。写死会僵。

## 皮肤与表情

- **黑白/扁平**:能用抖动线、潮红、多汗等漫画符号,演**夸张情绪比 3D 更顺手**。重表情戏选这俩
- **3D**:表情偏收敛但质感足,适合封面/产品

## 验证记录(②嘴硬)

纯形容词 → 心虚弱;纯情境描述 → 反而灿笑(萌脸);**情境+演技锚点 → 心虚到位**(八字眉+龇牙+汗)。三皮肤双图法全通过,黑白版表情最强。
