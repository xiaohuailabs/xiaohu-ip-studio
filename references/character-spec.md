# 角色包规范:怎么建一个自己的角色

本技能的角色是**可扩展**的:照本规范在 `characters/<名>/` 建一个目录,就进库,配图时能选它。

## 一个角色包 = 一个目录

```
characters/<名>/
├── character.md        # 角色规格(必须)
└── refs/
    └── <名>-锚点.png   # 形象锚点图(必须;生图时传它 --reference 锁形象)
```

## character.md 该写什么

照库里现成角色(如 `characters/lengjiao/character.md`)的结构填:

1. **角色包元信息表**:名称/英文、基础造型、主色、一句人设、默认画风、招牌识别符号、锚点图路径
2. **外形**:招牌识别符号(跨画风不变的那几样),逐条列
3. **性格**:一句话人设 + 适合配什么话题
4. **表情随情绪变**:这个角色怎么演不同情绪(默认表情 + 几种变体)
5. **英文 prompt 段**:注入生图的角色描述(`{表情}{动作}` 占位符,按本张替换)

## ⛔ 反复杂护栏(角色一致性的命根)

一个角色 = 一组**锁定**的简单选择,越简单跨图越稳:

- **一个简单轮廓**:一个在任何尺寸都认得出的基础形状(三角 / 圆 / 方 / 一个小动物)。复杂造型生成时会漂。
- **锁定的脸**:表情可变,但"几只眼、有没有眉/嘴"要写死、每张一致。脸是最先漂的地方。
- **一个招牌识别符号**:一眼认出它的那个标志(棱角的怒眉、牛马的牛角马脸、电量的电量格)。锁住它。
- **没锁的东西不许出现**:character.md 没写明的部件,生成时多出来了就重生。

判据:**把角色从画面里抹掉,如果图还讲得通,它就只是贴纸**——重画到"少了这个角色这事就发生不了"。

## 怎么生锚点图

1. 写一个线稿风 prompt(套 `prompt-template.md` + `style-dna.md` 的线稿块 + 你角色的英文描述);
2. 跑 `python3 scripts/generate.py --prompt-file <p.md> --out characters/<名>/refs/<名>-锚点.png`;
3. 不满意改 prompt 重生(失败最多 2 次)。

锚点图 = 这个角色配图时传 `--reference` 锁形象的那张。情绪幅度大的角色可以多生几张"演技锚点",配图换表情更稳。
