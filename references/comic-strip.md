# 四格漫画:用故事讲过程/转折

> 2026-05-31 加(第三轨,与情绪图/解释图并列)。小互的原生形态——"摸鱼记"本身就是连环画。讲"前因后果 / 踩坑到解决 / 过程演变 / 翻转"这种**有时间线**的内容,4 格比单张强一个量级。

## 何时用四格(而非单张)

信号:文章这段有**时间线 / 转折 / 心路历程**——"以前 X → 后来 Y""踩了坑 → 找到解法""从 A 演变到 B"。单张讲不出"过程",4 格能。
- 没时间线、就一个状态/结构 → 单张(情绪图 or 解释图)
- 有起承转合 / 前后变化 → 四格漫画

## ⛔ 配四格前先过闸:四条硬判据 + 负向自检(2026-06-15 补,借 illo composition)

四格容易**过度用**(只要"有点过程"就画四格)。配 panel 前先过两关:

**四条硬判据(满足其一才配 panel)**:①Causality 因果是主张(X 导致 Y,不先看 X 则 Y 读不出)②Accumulation 累积(步骤复利、越滚越多)③Turn 末格反转(最后一格落点)④Rhythm 节奏(同场景每格一变,pattern 本身是信息,如重试循环 / 开不完的会)。

**负向自检(否决)**:
- 格子**可以重排、或删掉任一格而不丢意义** = 它不是序列,塌回单张;
- 两态对比但**没有中间旅程** = 那是单张 change(对比图),不是四格漫画。

命中负向 → 别画四格,降级单张。(跟下面「何时用四格」是收紧关系:老信号"有时间线/转折"是门槛低的初筛,这四条是防过度用的复筛)

## 结构:起承转合(表情必须跟着递进)

- **格1 起**:开局 / 期待 / 现状
- **格2 承**:冲突 / 踩坑 / 崩溃(情绪最低点)
- **格3 转**:转机 / 发现解法(情绪反转)
- **格4 合**:结局 / 真香 / 躺平(情绪最高点)

小互在一张图里演完整**情绪弧**:期待 → 崩溃 → 惊喜 → 惬意。情绪起伏越大越好看。

## 技术要点

- **一张图 2x2 网格**(不是 4 张拼!),清晰格子边框 + 留白 gutter
- aspect `1:1`(2x2 方版)或 `3:4`
- **传长相锚点**(`characters/xiaohu/refs/xiaohu-ip-正面挥手.png`)保跨格一致;四格四情绪靠 prompt 分格描述,**不传单一演技锚**(否则四格一个表情)
- prompt 明写 "the SAME character across all 4 panels, expression PROGRESSES with the story",再逐格写清场景 + 情绪 + 中文标注
- 每格一句短中文标注
- 画风皮肤任选,3D 最像连环画

## 验证(Routines 踩坑真香,一次成)

格1 期待「想让它自己定时跑」(思考泡 AI+时钟)→ 格2 崩溃「自己搭全是破事」(流泪+ERROR+乱线,狐狸跟着哭)→ 格3 惊喜「敲一行 /schedule」→ 格4 躺平「它自己跑我躺平」(喝茶,狐狸睡腿上)。四格连贯、小互跨格一致、表情递进完整、中文零错字。
