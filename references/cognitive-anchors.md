# 认知锚点:该配图的点怎么挑

## 总原则:不要平均配图

普通配图工具每隔几段塞一张,图跟内容没关系,纯凑数。本技能反过来:**先把文章读透,只在"认知锚点"上配图**——读者会在这里停下来想一下、或者容易卡住的地方。没有认知动作的段落,不配图。

够用就好,别把正文做成画册。

## ⛔ 挑锚点前先做的判断:选对点 > 画得好(2026-06-15 补,借 illo composition)

挑哪里配图之前先过两道,防"挑了个好画但次要的细节配成主图,文章真正的论点反而没配上":

**① 三层分离**:把要画的点拆成——①修辞任务(作者想让你信什么)②核心主张 hero thesis(归成一句话)③支撑细节(证明它的具体例子 / 轶事)。⚠️ **最好画的那个瞬间,通常是③支撑细节,不是②核心主张——好画的画面常是诱饵**。先确认你要配的是②不是③。

**② 体裁制导**(每种文体该 hero 什么):
- 产品发布 → hero 新角色 / 能力跃迁(不是 logo)
- 复盘 / 事故 → hero 失败的前提 或 断掉的那个环
- 引文 / 观点金句 → hero 引文命名的那层抽象关系
- 教程 / How-to → hero 它带来的转变(前后差),不是步骤截图
- 跑分 / Benchmark → hero 跨过的那个阈值 / 意义,不是图表本身
- 故事 / 轶事 → hero 那个顿悟瞬间
- 评论 / 主张 → hero 主张的后果 或 它推翻的旧认知

判据:配图前对一下——这张图 hero 的,是这段文体真正的修辞任务,还是个好画的支撑细节?(跟 deep-reading Q0 文体→hero、anti-ppt-qa source-fit 复检呼应,这里是更靠前的"选对点")

## 认知锚点清单(挑这些)

- **核心判断**:作者下的一个反直觉结论。例:"拐点不是模型更聪明,是它可信到你敢离开座位"
- **两个断点**:一个流程里最容易断/最关键的两处
- **输入输出闭环**:A 进 → 处理 → B 出 的回路
- **分流**:一个东西被分到几条路(信任/转化/流量)
- **前后对比**:混乱→有序、手动→自动、分散→收拢、焦虑→稳定
- **一鱼多吃**:一份素材榨成多种用途
- **承接路径**:从想法到上线、内容怎么接住读者
- **常见坑**:大家都会踩的那个错
- **角色状态变化**:卡住→跑起来、工具太多→焦虑

## 按小互四品类配图侧重

小互写作分四品类(见 writing-guide),配图侧重不同:

| 品类 | 配图挑什么 | 别配什么 |
|---|---|---|
| **产品动态** | 这功能解决的那个具体场景痛点 / 用前用后对比 | 不配功能清单图(那是 PPT) |
| **教程** | 关键步骤的"为什么这么做" / 容易卡的那一步 | 不逐步截图(教程本身有步骤) |
| **深度解读** | 核心判断 / 机制隐喻 / 反直觉那一下 | 不配论文结构图 |
| **方法拆解** | 框架分层 / 承接路径 / 一鱼多吃 | 不配金字塔模型图 |

## 判断:这段到底要不要图

问自己三句:
1. 这段有没有一个"认知动作"(判断/转折/分流/对比)?没有 → 不配
2. 这个认知动作,文字已经说透了吗?说透了且不抽象 → 不配,图是浪费
3. 它能不能换成一个**物理画面**?换不出来 → 大概率不该配图,或者得再想隐喻(见 metaphor-method.md)

三句都过,才进 shot list。

## 进了 shot list,这张要哪类图?

再判一刀:读者在这卡住,是"没共鸣"还是"没看懂结构"?
- 要**态度/共鸣/钩子** → 情绪锚点图(小互演情绪,见 metaphor-method + expression-method)
- 要**看懂流程/组成/对比/进阶/关系** → 解释图示(见 explanatory-diagrams)

信号:文章在讲"一串步骤 / 几个组成 / 两种对比 / 渐进过程 / 谁触发谁"=该上解释图。

⛔ **看目的不看长相(2026-06-19 加,根治"功能图错当角色戏画大"):** 判轨问「**这张要读者"懂个事"还是"有个感觉"?**」——**懂事(讲清一个功能 / 流程 / 步骤)= 解释轨**,即便画面只有一个角色在演那个动作,角色也走解释图的"小而嵌入";**只为有感觉 / 共鸣 = 情绪轨**,角色才大。最易栽的坑:"单角色演一个功能"长得像情绪图/角色戏,活儿却是讲清功能 → **归解释轨、角色小**,别被"它像角色戏"骗去画大。后果直接影响角色占比(解释轨小 / 情绪轨大,见 SKILL 步骤2「角色占比」列)。

**还有第三种**:这段有**时间线 / 转折 / 心路历程**吗("以前 X→后来 Y""踩坑→解决")?有 → **四格漫画**(见 comic-strip),一张图讲完一个故事,比单张强。

一篇好配图 = 情绪图(钩子)+ 解释图(难懂处)+ 四格漫画(有故事处)三种搭配,按内容挑形态,**别全是小互演情绪**。

## ⛔ 分布 gate(出 shot list 后自检,SKILL 1.5 的门槛在这)

这是**否决项,不是配额**(别读成"每类来一张")——只拦"形态选错"和"全是情绪图凑数",**不强制任何类型的最小张数**:

- **否决·全是情绪图**:情绪锚点图占一半以上 → 多半是没分流、在用小互演情绪凑数,打回重判(全篇确实只需钩子、没难懂结构时例外,但要说得出理由)。
- **否决·形态选错(该解释图用了情绪图)**:某点是"几个组成 / 一串流程 / 对比 / 进阶"且文字没说透,却画成小互演表情 → 换解释图。⚠️ **不是"有结构就必须配 1 张解释图"——文字已说透就不配**。
- **否决·形态选错(该四格用了单张)**:某段是完整时间线 / 转折,单张讲不出过程 → 用四格。⚠️ 同样**不是"有时间线就必须来一张四格"**。
- **核心**:gate 只拦上面两类,一篇可以只有 1 张解释图、0 情绪图、0 四格——只要那张是内容真需要的。
