# Credits & Lineage · 致谢与血统

这个技能站在几个开源配图技能的肩膀上。**方法论是借鉴并改造来的;角色阵容和手绘线稿画风是本项目原创。** 借鉴学的是底层原理,不照抄外观。

## 方法论血统(致谢)

- **[helloianneo/ian-xiaohei-illustrations](https://github.com/helloianneo/ian-xiaohei-illustrations)**(小黑 · Ian)— 方法骨架来源:认知锚点筛选、原创隐喻三步法、反 PPT 负向防线、reference 分层按需读。本项目"挑锚点 → 编隐喻 → 反 PPT"的主流程脱胎于此。
- **[JimLiu/baoyu-skills · baoyu-article-illustrator](https://github.com/JimLiu/baoyu-skills)**(宝玉)— 文字渲染铁律(色值不入图、错字改 prompt 重生不代码涂改)、解释图结构化字段骨架、配图密度旋钮。
- **[dososo/juju-content-illustrations](https://github.com/dososo/juju-content-illustrations)**(卷卷)— 一篇一世界(叙事空间统一)、语义批注色表、同篇变化 4 维量化、线型语义、画外手势克制规则。
- **[tmchow/illo-skill](https://github.com/tmchow/illo-skill)**(Trevin Chow · illo-skill.com)— **顶层架构**:角色/画风双轴 + 方法论常量化("the methodology is the constant; the character and style are the parameters");文体→hero 守则、source-fit 选点复检、结构完整性 QA、角色包机制,以及 config / 后端 / 可安装这套开源工程范式。
- **[orange2ai/orange-line-illustration](https://github.com/orange2ai/orange-line-illustration)**(橙线 · orange2ai)— **展示模式 B1「极简场景」**方法来源:人极小物极大的尺度戏剧、唯一单点强调色、纯白大留白、右下角灰色金句叙事。⚠️ 该项目闭源商用需授权;本库只借方法、演员用自己的符号角色,不复刻其"小橙/线人/线猫"外观。见 `references/display-modes.md`。
- **[helloianneo/ian-xiaohei-scenes](https://github.com/helloianneo/ian-xiaohei-scenes)**(小黑 2.0 · Ian)— **展示模式 B2「真实物件现场」**方法来源:白底 + 手绘符号角色 + 照片级写实 3D 物体物理互动 + 留白叙事(对应本库皮肤 G)。同样只借方法不抄"小黑"外观。

## 本项目自己的部分

- **31 个 IP 角色**(手绘线稿 15:职场态 8 + 当代情绪态 7;谐音梗 meme 16:食物拟人 11 + 符号成精 5),所有角色设计、隐喻、画风原创。
- **中文深度文配图方法论**:逐节枚举计划表、双向第一性(天花板防凑数 + 地板防漏配难懂机制)、深层提炼三问(真意/张力/灵魂话)+ Q4 必现内容锁定、三轨分流(情绪锚点 / 解释图 / 四格漫画)、视觉契约六维统一。
